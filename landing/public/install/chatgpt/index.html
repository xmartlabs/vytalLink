What to expect

This page works whether ChatGPT supports installed apps for you already or still falls back to manual auth.

1

Keep the vytalLink app open

If you came here from the mobile app, your Word + PIN were prepared before this page opened. If you started on the web, open vytalLink first and tap Get Word + PIN.

2

Open ChatGPT through the install flow

ChatGPT may show an install or connect prompt for vytalLink. If it does, continue there. If not, open the chat anyway and use the fallback instructions below.

Open ChatGPT install flow View full setup guide
3

Fallback: paste Word + PIN

If ChatGPT asks you to connect manually, paste the Word + PIN from the vytalLink app. That fallback remains valid even when the install prompt is missing.

Missing the app?

Install vytalLink first, then come back here and continue the ChatGPT flow.
